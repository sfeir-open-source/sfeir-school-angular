# Migrating from ngIf to @if block

**Before**

```angular2html
<div *ngIf="user$ | async as user">
  <!-- content to display if user exist -->
</div>
```

<!-- .element: class="big-code" -->

<br/><br/>

**After**

```angular17html
@if(user$ |async; as user) {
  <div>
    <!-- content to display if user exist -->
  </div>
}
```

<!-- .element: class="big-code" -->
