# Syntax binding
<b>Events</b>
```html
<button type="button" (click)="changeName()">Refresh</button>
```
<!-- .element: class="medium-code"-->
<br>

<b>Properties</b>
```html
<button type="submit" [disabled]="myForm.invalid">Submit</button>
```
<!-- .element: class="medium-code"-->
<br>

<b>Reference</b>
```html
<input #myName type="text" />
<span>{{ myName.value }}</span>
```
<!-- .element: class="medium-code"-->
<br>

<b>Interpolation</b>
```html
<span>{{ name }}</span>
```
<!-- .element: class="medium-code"-->
