# Application Bootstrapping

Clone the project:

```bash
git clone https://github.com/sfeir-open-source/sfeir-school-angular.git
```

<br/>

At the project root:

```bash
npm install || npm ci
```

<br/>


```bash
npm run client -- "workshop_name"
```

<br/>


```bash
npm run server:start
```
