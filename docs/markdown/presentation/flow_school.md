# Bootstrapping de l'application

Clonez le projet:

```bash
git clone https://github.com/sfeir-open-source/sfeir-school-angular.git
```

<br/>

A la racine du projet

```bash
npm install || npm ci
```

<br/>


```bash
npm run client -- "nom_du_workshop"
```

<br/>


```bash
npm run server:start
```
